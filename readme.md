# 🌐 Anthropotech

> Un espace à l'intersection de l'anthropologie et de la technologie.

## 📖 À propos

**Anthropotech** est un site web dédié à l'exploration des liens entre les sciences humaines et les nouvelles technologies. Articles, analyses et réflexions sur la façon dont la tech transforme nos sociétés, nos cultures et nos comportements.

## 🗂️ Structure du projet

```
Anthropotech/
├── index.html        # Page d'accueil principale
├── articles/         # Dossier contenant les articles
└── README.md         # Ce fichier
```

## 🚀 Lancer le projet en local

1. Clone le repository :
   ```bash
   git clone https://github.com/shamsetdean/Anthropotech.git
   ```

2. Ouvre `index.html` dans ton navigateur — aucune dépendance requise.

## 🌍 Déploiement

Le site peut être hébergé gratuitement via **GitHub Pages** :

1. Va dans **Settings** → **Pages**
2. Sélectionne la branche `main` et le dossier `/ (root)`
3. Ton site sera accessible à `https://shamsetdean.github.io/Anthropotech`

## ✍️ Contribuer

Les contributions sont les bienvenues ! Pour proposer un article ou une amélioration :

1. Fork le projet
2. Crée une branche (`git checkout -b feature/mon-article`)
3. Commit tes changements (`git commit -m 'Ajout: nouvel article'`)
4. Push (`git push origin feature/mon-article`)
5. Ouvre une Pull Request
